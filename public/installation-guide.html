1. Prerequisites

Before you begin the installation process, ensure you have:

  • Access to your hosting control panel (cPanel, Plesk, etc.)
  • FTP account details (if you plan to use FTP for uploading)
  • Your domain name properly pointed to your hosting
  • The vishal-decor-installation.zip file

Tip: Make a backup of your existing website files if you're replacing an existing website.

2. Preparation

Extract the vishal-decor-installation.zip file on your computer. You should see:

  • A dist folder containing all website files
  • This installation guide

The dist folder contains all the files you need to upload to your hosting account.

3. Uploading Files

Using FTP (FileZilla, etc.)

  1. Connect to your hosting using your FTP credentials
  2. Navigate to the public HTML folder (usually called public_html, www, or htdocs)
  3. Upload all contents of the dist folder to this directory
  4. Make sure the .htaccess file is uploaded (it might be hidden by default)

Using Hosting File Manager

  1. Log in to your hosting control panel
  2. Find and click on the File Manager tool
  3. Navigate to the public HTML folder
  4. Click the "Upload" button and select all the files from the dist folder
  5. If you need to upload many files, consider creating a ZIP file, uploading it, and then extracting it on the server

4. cPanel Installation

  1. अपने cPanel अकाउंट में लॉग इन करें
  2. "File Manager" आइकन पर क्लिक करें
  3. public_html फोल्डर में जाएँ
  4. यदि पुरानी साइट बदल रहे हैं, तो पुरानी फाइलों का बैकअप लें या उन्हें रीनेम करें
  5. "Upload" पर क्लिक करें और अपने dist फोल्डर से सभी फाइलें सिलेक्ट करें
  6. वैकल्पिक रूप से, ZIP फाइल अपलोड करें और उसे public_html डायरेक्टरी में एक्सट्रैक्ट करें
  7. जांचें कि .htaccess फाइल सही तरीके से अपलोड हुई है

महत्वपूर्ण: यदि .htaccess फाइल नहीं दिख रही है, तो आपको फाइल मैनेजर सेटिंग्स में "Show Hidden Files" को इनेबल करना होगा।

5. Plesk Installation

  1. अपने Plesk कंट्रोल पैनल में लॉग इन करें
  2. "Websites & Domains" पर जाएँ और अपना डोमेन ढूंढें
  3. "File Manager" पर क्लिक करें
  4. httpdocs फोल्डर में जाएँ
  5. "Upload" बटन का उपयोग करके dist फोल्डर से सभी फाइलें अपलोड करें
  6. सुनिश्चित करें कि .htaccess फाइल सही तरीके से अपलोड हुई है

6. Installing in a Subfolder

यदि आप Vishal Decor वेबसाइट को सबफोल्डर में इंस्टॉल करना चाहते हैं (उदाहरण: yourdomain.com/vishaldecor):

  1. अपने होस्टिंग के पब्लिक डायरेक्टरी में सबफोल्डर बनाएं
  2. dist फोल्डर से सभी फाइलें इस सबफोल्डर में अपलोड करें
  3. आपको .htaccess फाइल को सबफोल्डर में काम करने के लिए संशोधित करना पड़ सकता है:
RewriteBase /your-subfolder-name/

इस लाइन को .htaccess फाइल में RewriteEngine On लाइन के बाद जोड़ें।

7. Troubleshooting

Common Issues and Solutions

404 Page Not Found Errors

यदि आपको साइट नेविगेट करते समय 404 एरर मिल रहे हैं:

  • सुनिश्चित करें कि .htaccess फाइल सही तरीके से अपलोड हुई है
  • जांचें कि आपका सर्वर mod_rewrite का समर्थन करता है (अपने होस्टिंग प्रदाता से पूछें)
  • यदि सबफोल्डर में इंस्टॉल कर रहे हैं, तो सुनिश्चित करें कि आपने RewriteBase निर्देश जोड़ा है

Missing Images or Styles

यदि आपकी साइट पर इमेज या स्टाइलिंग गायब है:

  • सुनिश्चित करें कि सभी फाइलें पूरी तरह से अपलोड हुई हैं
  • विशिष्ट त्रुटियों के लिए ब्राउज़र कंसोल जांचें (अधिकांश ब्राउज़र में F12 दबाकर)
  • फाइल अनुमतियों को सत्यापित करें (फाइलों का 644, डायरेक्टरीज का 755 होना चाहिए)

Server Error 500

यदि आपको इंटरनल सर्वर एरर मिल रहे हैं:

  • अपने सर्वर के एरर लॉग देखें
  • सिंटैक्स एरर के लिए अपनी .htaccess फाइल जांचें
  • सहायता के लिए अपने होस्टिंग प्रदाता से संपर्क करें

8. Maintenance and Updates

भविष्य में अपनी वेबसाइट अपडेट करने के लिए:

  • अपडेट करने से पहले हमेशा अपनी मौजूदा साइट का बैकअप लें
  • आवश्यकतानुसार फाइलों को नए संस्करणों से बदलें
  • यदि आपने इसे संशोधित किया है तो अपने कस्टमाइज्ड .htaccess की एक कॉपी रखें

सुझाव: अपलोड करने के बाद अपने इंस्टॉलेशन की जांच करने के लिए शामिल check-installation.html फाइल का उपयोग करें। बस अपलोड के बाद yourdomain.com/check-installation.html पर जाएँ।
