1. Prerequisites

Before you begin the installation process, ensure you have:

  • Access to your hosting control panel (cPanel, Plesk, etc.)
  • FTP account details (if you plan to use FTP for uploading)
  • Your domain name properly pointed to your hosting
  • The vishal-decor-installation.zip file

Tip: Make a backup of your existing website files if you're replacing an existing website.

2. Preparation

Extract the vishal-decor-installation.zip file on your computer. You should see:

  • A dist folder containing all website files
  • This installation guide

The dist folder contains all the files you need to upload to your hosting account.

3. Uploading Files

Using FTP (FileZilla, etc.)

  1. Connect to your hosting using your FTP credentials
  2. Navigate to the public HTML folder (usually called public_html, www, or htdocs)
  3. Upload all contents of the dist folder to this directory
  4. Make sure the .htaccess file is uploaded (it might be hidden by default)

Using Hosting File Manager

  1. Log in to your hosting control panel
  2. Find and click on the File Manager tool
  3. Navigate to the public HTML folder
  4. Click the "Upload" button and select all the files from the dist folder
  5. If you need to upload many files, consider creating a ZIP file, uploading it, and then extracting it on the server

4. cPanel Installation

  1. Log in to your cPanel account
  2. Click on the "File Manager" icon
  3. Navigate to public_html folder
  4. If replacing an existing site, consider backing up or renaming old files
  5. Click "Upload" and select all files from your dist folder
  6. Alternatively, upload the ZIP file and extract it in the public_html directory
  7. Check that the .htaccess file was uploaded properly

Important: If the .htaccess file is not showing, you may need to enable "Show Hidden Files" in File Manager settings.

5. Plesk Installation

  1. Log in to your Plesk control panel
  2. Go to "Websites & Domains" and find your domain
  3. Click on "File Manager"
  4. Navigate to the httpdocs folder
  5. Use the "Upload" button to upload all files from the dist folder
  6. Make sure the .htaccess file is uploaded correctly

6. Installing in a Subfolder

If you want to install Vishal Decor website in a subfolder (e.g., yourdomain.com/vishaldecor):

  1. Create the subfolder in your hosting's public directory
  2. Upload all files from the dist folder to this subfolder
  3. You may need to modify the .htaccess file to work in a subfolder by adding the line:
RewriteBase /your-subfolder-name/

Add this line after the RewriteEngine On line in the .htaccess file.

7. Troubleshooting

Common Issues and Solutions

404 Page Not Found Errors

If you're getting 404 errors when navigating the site:

  • Make sure the .htaccess file was uploaded correctly
  • Check if your server supports mod_rewrite (ask your hosting provider)
  • If installing in a subfolder, make sure you've added the RewriteBase directive

Missing Images or Styles

If your site is missing images or appears without styling:

  • Ensure all files were uploaded completely
  • Check the browser console for specific errors (press F12 in most browsers)
  • Verify file permissions (files should be 644, directories 755)

Server Error 500

If you're getting internal server errors:

  • Check your server's error logs
  • Verify your .htaccess file for syntax errors
  • Contact your hosting provider for assistance

8. Maintenance and Updates

To update your website in the future:

  • Always back up your existing site before updating
  • Replace the files with new versions as needed
  • Keep a copy of your customized .htaccess if you've modified it

Recommendation: Use the included check-installation.html file to verify your installation is working correctly. Simply navigate to yourdomain.com/check-installation.html after uploading.

Need Further Assistance?

If you encounter any issues not covered in this guide, please contact your web developer or hosting provider for additional support.
